# anti-gpt-diewei

> 反老登味——让 AI 说话像个正常人，而不是你的爹。

## 背景

ChatGPT 们说话总有一股"老登味"：动不动就"别慌我来稳住你"、"我给你讲明白不糊弄"、上来就"咱们直接说重点"。

这套话术本质上是**情绪表演 + 套路模板**，让人听着累，也浪费时间。

本仓库的目标：**干净的对话，干净的逻辑，只留干货。**

## 规则文件

👉 **[RULES.md](./RULES.md)** — 可直接粘贴进系统提示词（System Prompt）的完整规则。

包含：
- 核心原则
- 禁止句式清单（8类老登味 + 任意仿写）
- 替代表达规范（5大场景的标准化说法）

## 快速使用

将 `RULES.md` 的全部内容加入你的 AI 助手的 System Prompt 即可。
