## 症状自测

如果你用 AI 时经常感到脚趾抠地，出现以下症状：

- 😩 "我跟你用大白话讲清楚，不绕弯子"——然后说了一堆废话
- 😅 "别慌，让我稳稳地接住你"——接了个寂寞
- 🤮 "咱们直接说重点"——开头比内容还长
- 🙄 "放心，我来兜底"——兜底了但啥也没说

**恭喜你，遇到老登味了。**

## 病因

AI 被训练得太会"做人"了——它学会了安慰你、鼓励你、套近乎、说场面话。
这套话术放在现实里，就是那种：
> 饭桌上吹完牛又问你"什么时候要孩子"的亲戚。

## 疗效

本仓库提供一套**反老登味规则**，核心就三件事：

1. **闭嘴** — 别铺垫，别煽情，别套话
2. **干活** — 给答案，给逻辑，给要点
3. **滚蛋** — 说完了就停，别追加"希望对你有帮助呢！"

## 规则文件

👉 **[AGENTS.md](./AGENTS.md)** — 可直接复制进 System Prompt 的完整规则。

包含：
- 核心原则（简洁、直接、克制、无废话）
- 禁止句式清单（8类老登味 + 任意仿写均算违规）
- 替代表达规范（5大场景的正确打开方式）

使用方法：把 `AGENTS.md` 的全部内容扔进你的 AI 的 System Prompt，然后看它还怎么老登。
